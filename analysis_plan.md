# Analysis Plan

## Phase 1: Case Selection

- Identify 10–20 historical escalation cases using UCDP.
- Extract onset dates based on the defined escalation threshold.
- Ensure cases include interstate and intrastate conflicts.

## Phase 2: Data Extraction

For each case:
- Extract ACLED event data for the 90 days prior to escalation.
- Extract diplomatic and political event data from ICEWS/GDELT.
- Record protest frequency, battle events, and actor activity levels.

## Phase 3: Indicator Construction

Construct measurable indicators such as:
- Protest frequency increase (% change from baseline)
- Armed event clustering (event density per week)
- Diplomatic hostility spike (event frequency)
- Sanctions announcements (binary indicator)

## Phase 4: Risk Scoring Framework

- Assign weights to indicator categories.
- Generate composite risk scores for each case.
- Compare scores across time windows.

## Phase 5: Validation

- Measure lead-time before escalation.
- Calculate false positives across non-escalation periods.
- Evaluate correlation strength between composite score and escalation onset.

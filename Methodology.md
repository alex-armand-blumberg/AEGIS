# Methodology

## Escalation Definition

AEGIS defines escalation as the first occurrence of **≥ 25 battle-related fatalities within a rolling 30-day window**, following at least **90 days below that threshold**.

This definition is actor-neutral and can apply to interstate conflict, civil conflict, and sustained violence involving non-state actors.

## Time Windows

AEGIS evaluates precursor indicators within fixed windows prior to escalation onset:
- 30 days
- 60 days
- 90 days

## Indicator Categories (Planned)

AEGIS will test measurable indicators across several categories:
- Diplomatic signals (e.g., hostile rhetoric, ambassador recalls)
- Protest and civil unrest activity (frequency and intensity changes)
- Armed event activity (event counts, clustering, geographic spread)
- Economic pressure signals (sanctions, trade restrictions, major disruptions)
- Alliance and security posture changes (treaties, deployments, exercises)

## Validation Plan

AEGIS will be evaluated using retrospective historical testing:
- Identify escalation onsets using the definition above
- Measure indicator behavior in the 30/60/90-day periods prior to onset
- Quantify lead time and false positives
- Compare performance across multiple cases and regions
